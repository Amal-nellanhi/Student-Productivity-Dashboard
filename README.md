# 🚀 Student Productivity Dashboard

Welcome to the Student Productivity Dashboard! This is a lightweight web application featuring a task manager, a Pomodoro timer, and a daily goal setter. 

## 🎯 Project Goal
This repository isn't just about building a dashboard—it's a **practice ground for learning Git and GitHub collaboration**. By picking up issues and contributing to this project, you will practice:
* Creating and managing branches.
* Writing clean, meaningful commit messages.
* Opening Pull Requests (PRs) and linking them to issues.
* Reviewing code and collaborating with others.

## 🛠️ Tech Stack
* **HTML5:** Page structure and layout.
* **CSS3:** Styling, animations, and Dark Mode.
* **Vanilla JavaScript:** DOM manipulation, timers, and `localStorage`.

## 👩‍💻 How to Contribute (Your Git Workflow)

Ready to claim an issue? Follow these exact steps to get your code merged!

**1. Clone the Repository**
Open your terminal and clone this repo to your local machine:
`git clone https://github.com/YOUR-USERNAME/Student-Productivity-Dashboard.git`

**2. Create a New Branch**
Never work directly on the `main` branch! Create a new branch named after the feature or bug you are fixing. 
`git checkout -b feature/dark-mode-neon` (or `bugfix/timer-logic`)

**3. Make Your Changes**
Open the code in your editor (like VS Code), write your code, and test it in your browser.

**4. Commit Your Changes**
Once your feature is working, stage and commit your code with a clear message:
`git add .`
`git commit -m "Add neon pink styling to dark mode toggle"`

**5. Push to GitHub**
Push your new branch up to the remote repository:
`git push origin your-branch-name`

**6. Open a Pull Request (PR)**
Go to the GitHub repository in your browser. You'll see a green button to "Compare & pull request". Click it, describe what you changed, and link it to the issue you solved (e.g., "Closes #2"). Wait for an instructor or peer to review and merge your code!
